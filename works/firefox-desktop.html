Firefox Browser Projects

Firefox is a free and open-source web browser available for Windows, macOS and Linux operating systems.

Project 1. Activity Stream - Browser Bookmark


Activity Stream for Firefox collects all the things you do in the browser; including downloads, bookmarks, history, and recommended articles for the users. Bookmarking is one of the key features in the browser. After conducting user interviews to understand their mental model and digging into the data, we were able to visualize a journey map showing how user save and manage their bookmarks. We separated the process of accessing bookmark into four parts: save, organize, revisit, and use alternatives. We proposed several concepts and mockups to enhance bookmark design based on the findings.


  • Company: Mozilla
  • Product Type: Desktop App
  • Time: May 2016 - Sep 2016
  • Role: UX designer
  • Team: 2 UX designer, 1 Visual designer
  • Responsibilities: Interaction Design, Journey Map, Prototype
